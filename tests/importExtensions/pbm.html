ProtoBuf Extension test case

To run these tests, put these files on a server, so that Tests holds the protos, and the root is set properly.

The files MUST be on a server because you can't load files via AJAX from a file:// url in the browser.

Then, click the button to see a simplified outline of the objects in the tree.

As of the time of writing this code, the zmq_media_player was returning a single field, named 'MediaPlayerPbm'. However, this is an error in parsing the extensions. ZmqMediaPlayerPbm has no fields. It extends MediaPlayerPbm with the additional field zmq_media_player:

message ZmqMediaPlayerPbm
{
  extend MediaPlayerPbm
  {
    optional ZmqMediaPlayerPbm zmq_media_player = 310;
  }
}
